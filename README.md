# 🏃 SPORTIX - Sports E-commerce Website

## 📌 Description

SPORTIX is a sports e-commerce website developed to provide customers with a convenient online shopping experience for sportswear, footwear, fitness equipment, and sports accessories.

The system allows users to browse products, search and filter items, manage shopping carts, apply discount vouchers, place orders, and interact with a modern and responsive user interface.

The project was built using **HTML5**, **CSS3**, **JavaScript (ES6)**, and **Bootstrap 5**, focusing on responsive design, user experience, and front-end development best practices.

The website also includes an **Admin Dashboard** that supports product management, order tracking, customer management, voucher management, and sales statistics.
